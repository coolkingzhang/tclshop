2015年
  • 1月 TCL通讯收购Palm品牌，借助Palm在北美地区强大的品牌识别度，将进一步提升TCL通讯在北美手机市场的竞争力。 TCL集团携手湖北银行、武商联集团、武商集团，共同创立湖北消费金融股份有限公司。 TCL集团调整现有产业及业务结构，将集团主要产业和业务及企业分别归属为产品业务领域，服务业务领域以及创投及投资业务（简称：三大业务领域、“7+3+1”十一个业 务板块）。 TCL集团、TCL多媒体、TCL通讯投资9000万元在香港成立合资公司，运营智能家庭项目，大力布局智能生态系统。
  • 2月 TCL集团入股天津七一二通信广播有限公司(“七一二”), 进入军用、警用、铁路、交通等行业专用通信设备，集群移动通信系统等市场。7月再次增持七一二股权。

  • 5月 TCL集团旗下的智能电视运营平台广州欢网科技有限公司正式引入腾讯投资作为战略投资者，继续推进腾讯与TCL TV+平台的全面合作。 TCL集团战略投资的全球播科技（北京）有限公司与中国电影科学技术研究所共同启动“电影智能家庭影院”试点项目，并首次大范围推广全球播“同步院线”的产品服务及理念。
  • 6月 武汉华星光电项目提前封顶，TCL完善面板产业链布局。 TCL参加2015年美国国际显示周SID大会，华星光电显示技术产品创多项“视界”之最。 TCL-IMAX娱乐有限公司在上海召开IMAX臻享影院媒体发布盛会。 TCL创投成为第一批进入德交所风险投资网络系统的两家中国创投机构之一。
  • 7月 7月28日，TCL集团正式实施股份回购，并将根据市场情况继续实施股份回购，维护全体股东的利益。 2015年《财富》中国500强企业名单发布，在中国经济增长放缓的情况下，TCL集团逆势增长，较去年上升7位，越居第55位。
  • 8月 TCL集团股份有限公司与中国电信集团公司共同宣布达成集团战略合作协议，携手开创“互联网+工业制造”新时代。 TCL凭借在员工培养体系、人才管理方面的成就获第六届“亚洲最佳雇主品牌奖”。
  • 9月 胡润研究院发布《2015胡润品牌榜》，TCL集团品牌价值115亿，涨幅比例达448%，位列十大增值品牌榜榜眼。在家用电器品牌价值排名中，由第七位跃升到第四位。 TCL集团“转型升级·鹰旅戈行”活动于瓜州举办点将台启动仪式。 2015（第21届）中国品牌价值报告公布，TCL品牌价值710.28亿元，持续保持增长，在中国电视机制造业第10次蝉联榜首。 9月28日，“2015 TCL秋季新品发布会” 、第二届TCL铁粉节在在深圳召开，向用户展示了“双+”战略转型的全新成就。
  • 10月 TCL集团董事长兼CEO李东生入选2015互联网时代最具影响力企业家20强。 TCL集团股权悄变，TCL集团董事长兼CEO李东生及其管理层已成为TCL第一大股东
  • 11月 第十一届中国优秀企业公民年会上，TCL集团荣获“2015中国优秀企业公民”、“TCL希望工程烛光奖”项目获得“2015中国企业公民优秀公益项目”荣誉称号。
  • 12月 12月14日，TCL再度携手乐视在深圳举行战略合作发布会，推动产业升级共谋全球电视市场。 12月21日，央视-索福瑞与欢网科技举行战略合作发布会召开，宣布共同开展智能电视大数据在电视收视领域的研究。

2007年

  • 2月 TCL与中国网球协会签署为期3年的战略合作协议，成为中国女子网球队的首席赞助商;
    TCL与中国网球协会签署为期3年的战略合作协议，成为中国女子网球队的首席赞助商。
  • 3月 为期四天的“2007年TCL高尔夫精英赛”在海南三亚落下帷幕，该项国际性赛事吸引了包括多名欧洲高球名将在内的156名职业选手参与:
    中国共产党TCL集团第四次代表大会在惠州召开，选举产生了以李东生为党委书记的TCL集团新一届党委班子。
  • 4月 TCL自主研发的“增强型液晶电视数字视频动态背光控制技术”通过国家高新技术成果鉴定，并成功实现技术转让。此技术属国内外首创，且达到国际领先水平。
  • 6月 TCL启动新品牌战略，“创意感动生活”（The Creative Life）为“TCL”品牌注入新的内涵。
  • 7月 TCL炫尚E72液晶电视荣获德国红点工业设计大奖——2007产品设计奖。
  • 10月 TCL入选《财富》“2007最受赞赏的中国公司”。
  • 11月 李东生董事长应邀参加“相信中国制造——CCTV中国经济年度人物四海论道”全球论坛法国站演讲。
  • 12月 TCL集团宣布对旗下成员企业进行业务重组，集团治理向投资控股型管理模式转变。自2008年1月1日起，多媒体、通讯、家电、部品四大产业集团以及房地产与金融投资业务群、物流与服务业务群共六大业务单元构成TCL全新阵容;
    TCL品牌价值首次突破400亿元大关，达到401.36亿，上涨近40亿，成为“全国电视机制造业第一”品牌;
    TCL集团董事长李东生和夫人魏雪捐款800万元设立“华萌基金”，用于资助农村贫困地区的教育事业。这是中国企业家在“希望工程”设立的首个个人公益基金;
    TCL集团高票通过定向增发方案，将募集资金不超过16.9637亿元人民币。
2006年
  • 3月 第三届TCL高尔夫精英赛在海南三亚挥杆，TCL品牌知名度和美誉度在全球得到进一步提升。
  • 7月 李东生总裁发表《鹰之重生》系列文章，为TCL新一轮的变革创新活动指明了方向。7月中旬，TCL集团150多名高管举行延安行活动，并在宝塔山下立下坚定变革决心、成就 全球领先的誓言。
  • 8月 TCL集团企业文化变革创新誓师大会召开，这是继1998年、2002年后，TCL集团在推行企业文化创新方面的又一里程碑。
  • 9月 TCL集团相继在惠州、广州、深圳举行成立25周年工作汇报会，并举办TCL集团第6届合唱音乐会，展示TCL 25年来取得的巨大成就
  • 10月 国庆黄金周期间，TCL启动“十一聆听行动”，李东生总裁和集团其他高管分赴海内外各大卖场，零距离倾听顾客需求，把握市场脉搏;
    TCL集团启动精“鹰”工程培养计划，落实“关注中层、打造高效能中层队伍”的人力资源策略。经选拔，100名中层管理者列入首批培养计划。
  • 11月 TCL集团财务有限公司开业，宣布了“创建具有国际竞争力的世界级企业银行”的宏伟愿景。
  • 12月 中国最有价值品牌揭晓，TCL品牌价值达362亿元，从2005年的第六名跃居第三;
    李东生总裁再次当选“中国最具影响力的企业领袖”。
2005年

  • 2月 李东生总裁当选“2004广东十大经济风云人物”。
  • 3月 高尔夫欧巡赛43年历史上第一个由中国企业冠名并赞助100万美元的“TCL高尔夫精英赛”在三亚圆满收杆。
  • 4月 中共中央政治局常委、中央纪委书记吴官正视察TCL内蒙王牌生产基地。
  • 5月 李东生总裁当选“中国十大创业领袖”。
  • 6月 TCL荣获2004年度“中国最受尊敬企业”;
    2005年中国电子信息百强揭晓，TCL位居第三。
  • 10月 广东省推出名牌战略，明确TCL领衔广东13家企业冲顶世界名牌。
  • 11月 2005年中国最有价值品牌评估揭晓，TCL以336.66亿元的品牌价值排名第六;
    TCL荣获由中华人民共和国民政部颁发的2005年度“中华慈善奖”，是此次唯一获奖的消费类电子企业。
  • 12月 TCL与法国罗朗格公司签署国际电工、楼宇科技资产转让协议。
2004年
  • 1月 原中共中央政治局常委、全国人大常委会委员长李鹏同志视察TCL移动通信公司和TCL王牌惠州生产基地；
    中国国家主席胡锦涛、法国总理拉法兰等两国政要出席成立TCL－汤姆逊电子有限公司的合同签约仪式;
    TCL集团在深圳证券交易所正式挂牌上市。
  • 2月 李东生总裁被国际著名杂志《财富》评为“2003年亚洲年度经济人物”。
  • 6月 TCL入选 “2003－2004年度中国最受尊敬企业”;
    TCL在广州举行人才招聘新闻发布会，宣布年内在全球招聘2200名具有国际化背景的中高级经营管理人才和研发人才;
    TCL获广东省科学技术奖励特等奖;
    广东最大50家企业排名榜和广东最大50家工业企业排名榜揭晓，TCL分别排名季军和亚军。

    TCL与法国汤姆逊合资组建的全球最大彩电企业TTE CORPORATION(简称TTE)在深圳隆重开业。

    菲律宾总统阿罗约在北京会见袁信成高级执行副总裁;
    TCL空调武汉工业园奠基;
    TCL（无锡）数码工业园在无锡国家高新技术开发区奠基;
    TCL通讯科技控股有限公司在香港联合交易所主板上市。
  • 10月 TCL阿尔卡特移动电话有限公司开业;
    中共中央政治局常委、国务院副总理黄菊在深圳参观TCL的产品并听取李东生总裁的工作汇报。
  • 11月 TCL集团和东芝电器营销株式会社（“TCM”）签署谅解备忘录，成立两家以电冰箱、洗衣机制造为主的制造公司和以销售为主的销售公司;
    TCL白家电与青岛威士电器合资，建立TCL电冰箱青岛工业园
  • 12月 李东生总裁当选美国《时代周刊》2004年全球最具影响力商界领袖;
    TCL（成都）数码工业园在成都高新技术产业开发区破土动工;
    李东生总裁当选CCTV2004中国经济年度人物。

  • 7月 广东省省长黄华华视察TCL，称赞“TCL的成功经验值得推广”;
    李东生总裁发表《龙腾寰宇，虎跃神州——通过“龙虎计划”建立TCL国际竞争力》的重要讲话。

  • 12月 中国最有价值品牌评比揭晓，“TCL”品牌价值为267.12亿元，名列第六。
2002年
  • 2012年 李东生总裁被评为2002年“中国经济年度人物”，并获“年度创新奖”;
    TCL手机在中国市场排名第三，位列国产手机首位;
    TCL集团副总裁、TCL移动通信有限公司总经理万明坚被评为“全国十大杰出青年”;
    中国最有价值品牌揭晓，TCL品牌价值187.69亿，位居全国第六。
  • 1月 全国人大副委员长邹家华视察TCL，并题词“为创建世界名牌企业而努力奋斗”。
  • 2月 中共中央政治局委员、广东省委书记李长春第二次视察TCL集团。
  • 4月 中共TCL集团召开第三次代表大会，选举了以李东生为党委书记的TCL集团第三届党委会;
    TCL集团股份有限公司宣告成立，选举产生了第一届董事会、监事会成员，李东生任董事长兼总裁
  • 7月 TCL集团召开企业文化创新千人大会，李东生总裁作题为《创建具有国际竞争力的新企业文化》的重要讲话。
  • 10月 TCL全资收购的德国施耐德公司在慕尼黑开业。
  • 11月 李东生总裁当选为中共十六大代表，参加中国共产党十六次全国代表大会。
2001年
  • 2001年 李东生总裁提出“变革创新、知行合一”作为2001年TCL集团的工作主题;
    集团首次对下属12家企业实行KPI绩效考核;
    TCL移动通讯全年销售手机130万台，销售收入突破30亿元，成长为集团又一重要经济增长点;
    TCL品牌以144.69亿元的品牌价值稳居全国第五位。
  • 1月 TCL集团兼并无锡永固电子公司，成立TCL数码科技（无锡）有限公司
  • 2月 TCL HiD299e型家庭信息显示器通过广东省科技成果鉴定,开创了我国彩电网络化的先河。
  • 4月 与台湾瑞智精密股份有限公司合作，成立TCL瑞智（惠州）制冷设备有限公司，进军空调压缩机行业。
  • 6月 集团成立“增强国际竞争力实施方案”工作组，着手制定集团中长期战略规划。
  • 9月 TCL集团在北京、广州、惠州三地隆重举行庆祝TCL成立20周年暨TCL20周年成果汇报会。
  • 11月 TCL成功举办第3届职工运动会。
  • 12月 TCL对TV、AV等家电信息相关产品进行资源整合，成立多媒体电子事业本部，形成一个全新的以各事业部为虚拟企业架构的管理模式。
1999年
  • 1月 TCL国际控股有限公司股票在香港成功上市;
    信息产业集团成立，微软公司前中国区总经理吴士宏女士出任总经理;
    TCL斥资控股翰林汇软件产业公司，大举进军信息产业;
    内蒙古TCL王牌电器有限公司宣告成立，原内蒙彩虹电视机厂无偿划拨给TCL经营，探索出一条国企改革新路;
    李东生总裁作了题为《我们的目标：创建世界级的中国企业》的报告，正式提出向世界级企业迈进;
    “TCL”商标被国家工商行政管理局认定为“中国驰名”商标
  • 3月 TCL移动通信有限公司成立。
1998年
  • 1998年 中国进出口银行与TCL签定“20亿元人民币出口卖方信贷一揽子授信协议”，为TCL开拓海外市场提供了有力的资金支持。
  • 5月 TCL与台湾致福集团合作，成立TCL致福电脑有限公司。
  • 10月 TCL金能电池有限公司成立。
1997年
  • 1997年 TCL集团公司调整企业结构，撤消三个专业集团，重组为TCL集团有限公司;
    TCL集团公司成为惠州市首家国有资产授权经营试点企业;
    TCL与河南美乐集团实现强强联合，成立河南TCL-美乐电子有限公司。
1996年
  • 1996年 TCL集团兼并香港陆氏公司彩电项目，开创国企兼并港资企业并使用国有品牌之先河;
    TCL集团被国家经贸委和中国人民银行列入300户重点发展企业集团之一。
1995年
  • 1995年 TCL集团公司改组，下设“通讯”、“电子”、“云天”三大集团;
    TCL电话获国务院发展研究中心市场经济研究所授予"中国电话大王"称号。
1994年
  • 1994年 率先推出国内第一部无绳电话。
1993
  • 1993年 TCL将品牌拓展到电工领域，成立TCL国际电工（惠州）有限公司;
    TCL电子（香港）有限公司成立;
    TCL通讯设备股份有限公司股票在深交所上市，是国内通讯终端产品企业中第一家上市公司。
1992
  • 1992年 研制生产TCL王牌大屏幕彩电，投放市场一炮走红。同时，着手导入CI系统，成为国内较早实施CIS的国有企业之一。
1991年
  • 1991年 在上海成立第一个销售分公司，随后又在哈尔滨、西安、武汉、成都等地建立销售分支机构，成为今天TCL全国性销售网络的雏形。
1989年
  • 1989年 TCL电话机产销量跃居全国同行业第一名并一直名列前茅。
1986年
  • 1986年 TCL商标在国家工商行政管理局商标注册;

1985年
  • 1985年 兴办中港合资的“TCL通讯设备有限公司。
1981年
