新技术、新产品
  • 2010年. TCL推出国内首款基于Android操作系统的智能电视通过省科技成果鉴定，达到国际先进水平。
  • 2009年. 推出全球首款3D商用电视
  • 2008年. TCL推出全球首台高清影像互动电视－X9
  • 2008年. 由TCL工业研究院独立研发的TCL闪联中间件平台通过科技鉴定并正式有偿授权给深圳纵横信科技有限公司，这是国内消费电子厂家首次实质性地完成了互通互联技术授权。
  • 2007年. TCL圆偏振光液晶电视技术通过国家鉴定，该技术属国际先进。
  • 2007年. 在中国闪联标准成为全球首个3C协同国际标准的同时，TCL推出首款具有鲜明家电特点的闪联盒子
  • 2007年. TCL增强型液晶电视动态背光技术通过国家鉴定，该技术属国内外首创，并成功授权日本瑞萨科技。
